## Этапы решения задачи оптимизации

1. **Идентификация задачи.**  
   Модель корректно определила тип задачи: условный экстремум с двумя линейными равенствами. Ограничения (x+y=2 и 3x-y=1) правильно интерпретированы как прямые, пересечение которых даёт единственную допустимую точку.

2. **Запись функции Лагранжа.**  
   Функция Лагранжа составлена правильно: к целевой функции добавлены оба ограничения с соответствующими множителями Лагранжа.

3. **Система уравнений первого порядка.**  
   Частные производные рассчитаны без ошибок, а уравнения ограничений переписаны верно. Модель корректно сформировала систему стационарных точек, учитывая оба ограничения одновременно.

4. **Поиск стационарной точки.**  
   Решение системы ограничений дало точку (x, y) = (3/4, 5/4). Поскольку ограничения линейные, это единственная допустимая точка. Ошибок на этом этапе нет.

5. **Нахождение множителей Лагранжа.**  
   Подстановка найденной точки в уравнения градиента Лагранжиана позволила корректно вычислить множители: λ₁ = 109/8, λ₂ = -15/8. Эти значения демонстрируют полный ход решения и помогают проверить правильность составления Лагранжиана.

6. **Вычисление значения целевой функции.**  
   Значение функции в точке (3/4, 5/4) вычислено правильно: f = 203/16 ≈ 12.6875. Поскольку это единственная допустимая точка, она автоматически является условным минимумом.

7. **Критерий Сильвестра.**  
   Гессиан целевой функции рассчитан верно. Модель корректно указала на положительную определённость матрицы: все ведущие миноры положительны, что подтверждает, что точка является минимумом. Таким образом, метод второго порядка полностью подтверждает найденный экстремум.

8. **Общий вывод по корректности решения.**  
   Все ключевые алгебраические шаги выполнены верно. Метод множителей Лагранжа применён корректно, стационарная точка определена точно, проверка через Гессиан подтверждает минимальное значение функции.
